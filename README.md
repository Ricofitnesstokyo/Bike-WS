# Site de coaching sportif

Site vitrine statique pour une offre annuelle de coaching sportif personnalisé.

## Offre
- Programme d'entraînement personnalisé sur 12 mois
- Programme quotidien avec mises à jour
- Suivi régulier
- Chat privé et groupe privé
- Contrôle hebdomadaire avec ajustements si besoin
- Tarif unique : 2790 € / an
- Option paiement : 4 x 697,50 €

## Fichiers
- `index.html` : structure du site
- `styles.css` : design responsive
- `script.js` : animations et FAQ

## À personnaliser
- Nom de marque
- Email de contact
- Mentions légales / CGV
- Lien de paiement Stripe, PayPal ou autre
- Connexion du formulaire à un outil comme Formspree, Tally, Typeform, Brevo ou Mailchimp
